Did you still remember it? If you forget it, let's learn 'Note'!

In Lava, 'Note' is line starts double slashes, use '//' to indicate.

Example:
	public class Has_slashes_hello
	{
		//Main
		public void main()
		{
			//Output to terminal
			System.screen.msg.create("Hello, World!");
		}
	}
